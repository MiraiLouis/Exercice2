# Exercice2

--------------------------------------------------------------------

L'objectif de ce programme est de simuler un ordonnanceur "premier arrivé, premier servi"

--------------------------------------------------------------------

-videFonction.c: initialise les liste des structures en tant que liste vide
-estVide.c: vérifie si la liste de processus est vide, dans ce cas renvoie 0, sinon renvoie 1
-step.c:Effectue un tour d'ordonnanceur
-run.c:execute step.c tant que le la file de processus n'est pas vide
-ajout_activite: permet d'ajouter un processus dans la liste de processus
-main.c: execute tous les programmes/porcédures cités ci dessus
